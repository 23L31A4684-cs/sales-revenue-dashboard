# Sales and Revenue Analysis Dashboard using Python
## Overview
This project analyzes sales and revenue data and provides insights using an interactive dashboard.
## Features
- Total sales and revenue tracking
- Region-based filtering
- Revenue trend visualization
- Product performance analysis
## Tools Used
- Python
- Pandas
- Streamlit
